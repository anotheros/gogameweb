# 棋盘识别需求

## 1.项目背景

需要在iOS 上进行一个拍围棋棋盘转换成二维数组的功能

视觉识别 openvc 棋盘棋子识别YOLO 都在ios终端上进行。

有人给做了个 python版 识别不准确。只能识别19*19 棋盘
